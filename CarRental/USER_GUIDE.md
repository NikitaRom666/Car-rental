# Користувацький гайд

## Як користуватися
1. Відкрийте термінал у папці CarRental.
2. Запустіть консольний застосунок.
3. Оберіть авто за номером або GUID.
4. Введіть дату початку і дату завершення у форматі `yyyy-MM-dd`.

## Правила
- Дата початку не може бути раніше ніж 2026-01-01.
- Дата завершення має бути після дати початку.
- Один автомобіль не можна бронювати двічі на той самий період.

## Повідомлення результату
- Успіх: бронювання створено, показано його ID.
- Помилка: застосунок пояснює причину українською мовою.
