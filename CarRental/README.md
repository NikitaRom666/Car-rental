# Оренда автомобілів

Навчальний капстоун про систему оренди автомобілів. Проєкт побудований як один репозиторій із шаровою архітектурою, бізнес-правилами, файловою персистенцією та тестами.

## Запуск
```powershell
dotnet run --project ConsoleUI/ConsoleUI.csproj
```

## Тестування
```powershell
dotnet test
```

## Основні артефакти
- бачення проєкту
- беклог
- діаграма класів
- діаграма послідовності
- діаграма архітектури
- звіт першої ітерації
- звіт другої ітерації
- звіт третьої ітерації
- тестова документація
- фінальний звіт
- демо-матеріали
- користувацький гайд
- довідник для розробника
- журнал змін
