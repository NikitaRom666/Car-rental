# Посібник користувача (User Guide)

## Кроки для успішного використання системи
1. **Перегляд доступних авто:** При запуску оберіть пункт "Показати автопарк". Доступні машини відображаються із зеленим маркуванням статусу.
2. **Бронювання автомобіля:** Введіть унікальний ID машини, вкажіть ID клієнта та задайте період оренди (кількість днів). Система автоматично підрахує та виведе фінальну вартість замовлення.
3. **Повернення автомобіля:** Після закінчення терміну оренди оберіть пункт "Завершити замовлення", щоб повернути автомобіль у загальний пул доступних транспортних засобів.
