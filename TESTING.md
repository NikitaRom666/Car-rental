# Звіт про тестування (Testing Documentation)

## Стратегія тестування
Для перевірки бізнес-логики застосовано фреймворк **xUnit**. Об'єктом ізольованого тестування виступає сервіс бронювання `BookingService`.

## Автоматизація
Тести налаштовані на автоматичний запуск у хмарі через GitHub Actions при кожному пуші. Локальний запуск здійснюється командою `dotnet test` після встановлення SDK.
