# Release Plan v1.0.0 (Car Rental System)

## Ѕсоре версії v1.0.0 (Що входить)
1. **Управління автопарком (Саг Management):** Повний CRUD (створення, читання, оновлення, видалення) для сутностей автомобілів. Підтримка метаданих: марка, модель, категорія (Економ, Бізнес, SUV), добова вартість, поточний статус доступності.
2. **Оформлення замовлень (Booking Engine):** Реєстрація клієнтів та формування контрактів на оренду. Перевірка часових проміжків на відсутність конфліктів накладання дат.
3. **Гнучке ціноутворення (Tariff Strategies):** Автоматичний динамічний розрахунок вартості із застосуванням паттерну "Стратегія" залежно від тривалості оренди та дня тижня.
4. **Збереження даних (Persistence):** Надійне локальне збереження всього стану системи у форматі JSON.
5. **Тестування:** Покриття бізнес-правил автоматичними тестами (>80% Code Coverage).

## Перенесено на майбутнE (Post-Course Backlog)
* Інтеграція із зовнішнім платіжним сервісом Stripe для верифікації транзакцій.
* Модуль автентифікації користувачів через JWT (JSON Web Tokens).
* Реалізація інтерактивної карти паркувальних зон автопарку.

## Допустимий технічний борг
* Використання файлової системи (JSON) замість повноцінного клієнт-серверного СКБД (наприклад, PostgreSQL).
* Логування помилок здійснюється безпосередньо у консоль/текстовий файл без інтеграції шини повідомлень або зовнішніх систем агрегації логів (Serilog/ELK).
