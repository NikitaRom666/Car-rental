# Extension Plan (СР 29)

## Цільові розширення для закриття прогалин
1. **Оптимізація пошуку характеристик автомобіля:** Інтеграція структури `HashSet<string>` замість стандартного списку рядків для зберігання додаткових опцій авто (наприклад, "GPS", "Autopilot", "Conditioner").
2. **Впровадження Generic Utility:** Створення узагальненого допоміжного класу або методів розширення для типізованої обробки та клонування доменних моделей при транзакціях бронювання.
