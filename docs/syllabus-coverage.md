# Матриця покриття тем навчального курсу

| Група тем курсу | Статус використання | Конкретне місце та сутність реалізації в Car Rental |
| :--- | :--- | :--- |
| **Основи ООП** | Повністю використано | Класи `Car`, `Booking`. Інкапсуляція закритих полів за допомогою C# властивостей (get; private set;). |
| **Абстракції та поліморфізм** | Повністю використано | Контракти репозиторіїв визначені через інтерфейси (`ICarRepository`). Поліморфна поведінка розрахунку вартості оренди. |
| **Generics, колекції, LINQ** | Повністю використано | Використання вбудованих у .NET узагальнених колекцій (`Dictionary<Guid, Car>`) та LINQ-запитів для фільтрації та вибору автомобілів. |
| **Обробка помилок та Persistence** | Повністю використано | Кастомні винятки `CarUnavailableException`. Серіалізація/десеріалізація в JSON-файли за допомогою бібліотеки `System.Text.Json`. |
| **Принципи SOLID** | Повністю використано | **SRP**: розділено збереження даних та бізнес-логіку. <br>**OCP**: гнучка зміна тарифних стратегій без модифікації коду сервісу. |
| **Об'єктні патерни** | Повністю використано | Патерн **Repository** (інфраструктура даних) та патерн **Strategy** (алгоритми розрахунку фінансової вартості оренди). |
| **Рефакторинг коду** | Повністю використано | Очищення стилю написання за допомогою утиліти .NET CLI `dotnet format`, усунення архітектурних code smells перед релізом. |
