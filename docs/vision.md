# Vision - Car Rental System
**Назва проєкту:** Car Rental System (Система оренди автомобілів)
**Опис проблеми:** Автоматизація обліку автомобільного парку, розрахунку вартості прокату та уникнення накладання дат бронювань.
**Цільові користувачі:** Менеджери автопарку, клієнти сервісу.
**Основні сценарії:** Реєстрація авто, створення бронювання з перевіркою доступності, повернення автомобіля з розрахунком пені.
**Нефункціональні вимоги:** Модульність (розділення на шари), висока тестованість бізнес-правил, стійкість до помилок файлового I/O.
