# Extension Report (СР 29)

## Результати впровадження розширень
1. **HashSet Оптимізація:** Завдяки переходу на `HashSet<string>` для перевірки комплектації автомобіля під час створення замовлення, часову складність валідації знижено з лінійної **O(M)** до константної **O(1)**.
2. **Generic Utilities:** Впроваджено кастомні методи розширення, що дозволяють безпечно мапити доменні сутності у моделі збереження інфраструктурного шару без дублювання коду для кожного типу окремо.
